# PassJava-Learning

> PassJava 学习教程，架构、业务、技术要点全方位解析。

PassJava 是一款帮助 Java 面试的开源系统，
可以用零碎时间利用小程序查看常见面试题，夯实 Java 基础。
采用流行的技术，如 SpringBoot、MyBatis、Redis、 MySql、
MongoDB、 RabbitMQ、Elasticsearch，采用 Docker 容器化部署。
